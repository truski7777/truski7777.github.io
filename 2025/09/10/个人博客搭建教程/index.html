GitHub Pages + Hexo 搭建个人博客

大家好，菜鸟第一次搭建自己的博客，记录一下搭建博客的过程和一些遇到的问题。

一、准备工作

1、注册Github账号

GitHub官网：https://github.com/

账号注册教程：https://blog.csdn.net/yaorongke/article/details/119086305

2、安装git
官方地址：https://git-scm.com/
GIt安装教程：https://www.cnblogs.com/linuxgeek/articles/18928378

安装好之后，在任意文件鼠标右键可以看到：Git Bash Here；

输入：git –version

出现这个说明git安装成功。

3、安装node.js
官方地址：https://nodejs.org/en

输入 node -v

出现这个说明nodejs安装成功

4、安装Hexo
安装命令：npm install hexo -g

测试是否安装成功，命令：hexo -v

5、安装hexo依赖
命令：npm install –save hexo-deployer-git

二、git配置SSH key

目的：可以免密的将本地的源码和资源上传到github，无需要每次都输账号和密码。

1、配置ssh

先生成ssh key，命令如下：
ssh-keygen -t rsa -C “邮件地址”
注意：这里的邮件地址是github账号绑定的邮件地址

可以看到在用户文件夹下生成了一个.ssh文件夹

进入到ssh文件夹，复制id_rsa.pub文件里的全部内容

接着打开github主页，点击个人设置，点击左侧的SSH and GPG keys，点击New SSH key，将id_rsa.pub复制的内容粘贴到key中，title随便起一个就行。

点击 Add SSH key

测试是否成功：ssh -T git@github.com

输入测试命令后，接着输入yes，出现下面这个提示说明成功

配置账号和密码

$ git config –global user.name 你的用户名
$ git config –global user.email 你的邮箱地址

在GitHub创建一个远程仓库

注意：

仓库名称格式 ：用户名.github.io

这个仓库名称将作为你github博客的访问地址

（推荐）有关git的详细教程可以看这个：https://juejin.cn/post/7245176801491894333

三、 搭建个人博客

1、新建博客

开始之前先介绍一下heox的基本命令

hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo deploy #部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本

在使用时可以使用缩写，如：
hexo n == hexo new
hexo g == hexo generate

2、新建一个保存博客的存放目录，比如D:\博客\0001

进入到本地博客存放目录，鼠标右键打开git
初始化个人博客：hexo init

可以看到博客存放目录出现了如下内容：

生成静态网页 ： hexo g
本地预览 ：hexo s

在浏览器访问：http://localhost:4000/

3、部署到github

编辑博客根目录下的 _config.yml 文件

deploy:
type: git
repository: 自己的Github仓库地址
branch: main

注意：这一部分在末尾，branch填写分支

同时还要修改

url:https://用户名.github.io

root:/

这个如果不修改，浏览器访问时是空白页面

发布到github ：hexo d

在浏览器访问
https://用户名.github.io/

4、编写博客

搭建好博客之后，日常中肯定是需要编写博客并且同步到github

新建一篇的命令为：hexo new ‘xxxx’

查看对应的文件夹

可以看到生成了一个个人博客部署.md文件，通过这个md文件去写博客

默认的内容是这样的：

title: chenge-first-blog
date: 2021-07-22 14:08:14
tags:

开始编写博客内容（markdown格式）

编写好之后：

清除本地缓存：hexo clean

生成html文件：hexo g
上传到github：hexo d

再次访问：

https://用户名.github.io/

四、更换主题

如果觉得hexo默认的主题不好看，可以根据个人喜好更换主题，官方提供了数百种主题供用户选择

官网：https://hexo.io/

每个主题都有相对应的用户文档，可以根据用户文档来配置

**注意：博客的根目录和theme文件夹中都有 ****_config.yml **配置文件，修改配置文件时注意所在目录

博客地址：https://truski7777.github.io/
